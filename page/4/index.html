Valine评论插件报错：Code 403
发表于2020-11-17|Hexo配置问题汇总
问题描述在使用 Hexo 搭建博客过程中，集成了 Valine 评论插件，本地使用时正常，将项目部署至静态网站后，Valine 插件在页面使用处提示 Code 403错误。 Code 403: Access denied by API domain white list, Please check your security domain. 大致意思是：访问被api域名白名单拒绝，请检查你的安全域名设置 解决方案在 LeanCloud 的【应用 > 设置 > 安全中心 > Web安全域名】中添加新的自定义域名即可解决该问题。 此处【新的自定义域名】指的是：你的博客的访问域名。 安全域名的具体添加方式根据 LeanCloud 的要求添加即可。
